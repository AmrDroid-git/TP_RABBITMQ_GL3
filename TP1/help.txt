0. Make sure to be in the right directory :

cd TP1





1. Compile all files

- Windows :
javac -cp ".;./lib/*" *.java
- Linux : 
javac -cp ".:./lib/*" *.java





2. Basic Queue Messaging (The "Hello" Queue)
- Send.java:
    Connects to RabbitMQ and drops a simple,hardcoded message ("7mema")
    directly into a specific queue named "hello".
- Receive.java:
    Connects to that exact same "hello" queue, waits for messages to arrive,
    and consumes them to print to your screen.

For the send and receive codes, just run them easily.

- Windows :
java -cp ".;./lib/*" Receive # in one terminal
java -cp ".;./lib/*" Send # in another terminal
java -cp ".;./lib/*" Send # repeat this us much as you want
- Linux :
// same commands, just replace ";" with ":"





3. Exchange Routing (The Logging System)
- EmitLogDirect.java:
    Sends a log message to a sorting center called an "exchange" and attaches
    a specific severity label (like `info` or `error`) to it, without knowing
    what queue it will end up in.
- ReceiveLogsDirect.java:
    Creates a temporary, nameless queue and tells the exchange,
    "Only route messages to my queue if they have these specific severity labels,"
    and then consumes those filtered messages.

For the emit and receive logs codes, you have to compile them manually then run
with args.

- Windows :

// Start the receiver (consumer) first in one terminal:
java -cp ".:lib/*" ReceiveLogsDirect info warning error
// Start the sender (producer) in another terminal:
java -cp ".:lib/*" EmitLogDirect error   "Help, the database is on fire!"
java -cp ".:lib/*" EmitLogDirect info    "Data has been sent!"
java -cp ".:lib/*" EmitLogDirect warning "The execusted command is deprecated!"
// You should be able to see the results in the first terminal

// Exit and repeat
java -cp ".:lib/*" ReceiveLogsDirect error
// In another terminal
java -cp ".:lib/*" EmitLogDirect info    "Data has been sent!"
java -cp ".:lib/*" EmitLogDirect error   "Help, the database is on fire!"
java -cp ".:lib/*" EmitLogDirect warning "The execusted command is deprecated!"
// Only the error should be received
